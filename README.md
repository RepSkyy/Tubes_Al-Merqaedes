# Tubes_AL-Merqaedes
Dalam proses pengembangan bot Robocode Tank Royale, program dikompilasi menggunakan bahasa pemrograman C# (.NET) melalui starter pack Robocode Tank Royale yang telah disediakan. Proses build dilakukan menggunakan command line dan .NET SDK agar setiap bot dapat dijalankan pada game engine Robocode.

Implementasi Algoritma Greedy:
Leopard21: Bot yang berfokus pada pemilihan target berdasarkan jarak terdekat.
LEnergyHunter: Bot yang menggunakan pendekatan greedy berdasarkan energi lawan.
botKedua: Bot yang berfokus pada sistem tracking target menggunakan radar lock dan predictive movement.
botDll: Bot sederhana yang menggunakan movement linear dengan radar sweep penuh secara terus-menerus.
